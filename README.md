## tn-skills-tasks

This repository contains all the tasks completed as part of my Data Science Internship. Each task focuses on different real-world data problems including data analysis, visualization, machine learning, and prediction models.

----

## Task 1: Customer Behavior Analysis & Segmentation
# 📌 Objective:

Analyze customer data and perform segmentation using clustering.

# 🔍 Key Steps:

Data Cleaning (handling nulls & duplicates)
Exploratory Data Analysis (EDA)
Visualization:
Age distribution
Gender distribution
Spend vs purchases
Customer Segmentation using K-Means Clustering

# 🛠️ Tech Used:

Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn (KMeans)
